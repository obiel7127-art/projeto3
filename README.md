O projeto usa um index.html com um título, um contêiner de slides e dois botões: anterior («) e próximo (»). 

As imagens do slideshow são carregadas dinamicamente dentro da div #container-items.

O arquivo slideShow.js controla a navegação: ele troca as imagens ao clicar nos botões previous e next.

O arquivo style.css organiza o layout, centraliza o slideshow e estiliza os botões.

Para usar: coloque todos os arquivos na mesma pasta e abra o index.html no navegador.
